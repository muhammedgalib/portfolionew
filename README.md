# 🎨 Graphic Design Portfolio

A premium, dark-themed personal portfolio website designed to showcase graphic design work, branding projects, and professional experience. Built with a focus on visual impact, smooth animations, and responsiveness.

**[🔗 View Live Demo](https://muhammedgalib.github.io/portfolio/)** *(Note: Ensure GitHub Pages is enabled in your Settings to see the live site)*

---

## ✨ Features

* **💎 Premium Dark Mode Design:** A sleek, modern aesthetic with neon blue accents and glassmorphism effects.
* **📱 Fully Responsive:** Looks perfect on desktops, tablets, and mobile devices.
* **⚡ Smooth Animations:** Integrated **AOS (Animate On Scroll)** library for engaging entrance effects.
* **🖼️ Lightbox Gallery:** Click on portfolio thumbnails to view high-resolution versions in a full-screen overlay.
* **⏳ Experience Timeline:** A vertical timeline showcasing professional history (SIBĀQ 2025, Annahda, AHSAn).
* **💬 Floating WhatsApp:** A pulsing, floating action button for instant client communication.
* **🛠️ Services Grid:** detailed list of services including Print Design, Branding, Packaging, and Photography.
